# Account Action Protocol

Use this when you want Codex to help with posting, replying, or managing launch actions.

## Safe Way To Let Codex Help

1. You log in yourself in Chrome or the in-app browser.
2. Do not share passwords, verification codes, recovery codes, API secrets, or ID documents.
3. Tell Codex exactly which platform to operate:
   - Reddit
   - Indie Hackers
   - Product Hunt
   - X / Twitter
   - Xiaohongshu
   - TikTok
   - Gmail
4. Codex can draft, navigate, paste, and prepare content.
5. Before any public post, DM, email, purchase, ad spend, or account change, Codex must show the final text/action and wait for your confirmation.

## What Codex Can Do After You Log In

- Open target URLs from `docs/outreach-targets.md`.
- Check community rules before posting.
- Draft replies and posts.
- Paste drafts into compose boxes.
- Help you edit for tone and compliance.
- Track actions in `docs/lead-tracker.csv`.
- Prepare Product Hunt listing copy.
- Prepare Reddit/Indie Hackers launch comments.

## What Codex Must Not Do Without Confirmation

- Send a post.
- Send a DM.
- Send an email.
- Spend money.
- Change account security settings.
- Delete content.
- Upload sensitive identity documents.
- Submit tax, legal, or payment onboarding forms.
